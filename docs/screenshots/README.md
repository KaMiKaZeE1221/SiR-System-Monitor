# Screenshot Guide

Place screenshots in this folder using these exact names so the root README renders them:

- `01-main-dashboard.png`
- `02-settings-sidebar.png`
- `03-sensor-selection.png`
- `04-summary-mode.png`
- `05-web-monitor.png`

Suggested capture settings:

- Format: PNG
- Width: 1600px or 1920px
- Theme: pick one and keep consistent across all captures
- Avoid sensitive information (IP addresses, usernames, hostnames) if sharing publicly

Quick capture checklist:

1. Launch app and set final theme.
2. Capture dashboard with multiple active sensor groups.
3. Capture grouped settings sidebar.
4. Capture Sensor Selection panel with reorder controls visible.
5. Capture Summary Mode.
6. Capture Web Monitor page in browser.
