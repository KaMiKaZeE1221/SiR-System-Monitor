Setup Guide

Quick summary: RTSS is mainly used for FPS and Frame Time. Most hardware sensors should come from AIDA64 shared memory.

1) AIDA64 setup (required)

  • Open AIDA64 and go to File → Preferences → External Applications.
  • Enable Shared Memory.
  • Select the sensor items you want exported/detected by this app.
  • Keep AIDA64 running while this monitor app is open.

2) RTSS / MSI Afterburner (optional)

  • Use RTSS primarily for FPS and Frame Time.
  • If FPS/Frame Time are missing, confirm RTSS is running and shared memory/OSD is enabled.

3) In this app

  • Leave Detection Mode on Shared Memory Mode.
  • In Shared Memory Sources, enable AIDA64 (and any other provider you use).
  • Use Sensor Selection to show/hide and reorder sensors.

4) If sensors are missing

  • Re-check that AIDA64 Shared Memory is enabled.
  • Confirm the exact sensors are selected in AIDA64 for external export.
  • Verify the corresponding source toggle is enabled in this app.
